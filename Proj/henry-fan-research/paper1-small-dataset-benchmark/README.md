# ML Small-Dataset Benchmark

> **Research Question:** On small tabular datasets (*n* < 1000), does deep learning 
> outperform classical ML in accuracy and F1 — and at what computational cost?

This is the full research pipeline for the paper:

**"Classical vs. Deep Learning on Small Tabular Datasets: A Reproducible Benchmark"**  
Henry Fan · San Jose State University · 2024

[![Python 3.9+](https://img.shields.io/badge/python-3.9+-blue.svg)](https://python.org)
[![scikit-learn](https://img.shields.io/badge/sklearn-1.8-orange.svg)](https://scikit-learn.org)
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

---

## Key Result (TL;DR)

> SVM and Logistic Regression match or beat neural networks on all four small datasets, 
> at 10–100× lower training cost. The Friedman test confirms significant differences 
> on 3/4 datasets (*p* < 0.05).

## Results Summary

| Dataset       | Best Model         | Accuracy | Train Time |
|---------------|--------------------|----------|------------|
| Iris          | RF / SVM           | 0.960    | < 1s       |
| Wine          | LR / SVM           | 0.983    | < 0.05s    |
| Breast Cancer | SVM                | 0.977    | < 0.02s    |
| Titanic       | SVM                | 0.908    | < 0.05s    |

Neural Network achieved the **lowest accuracy** on 3/4 datasets despite the highest architectural complexity.

---

## Statistical Tests

- **Friedman test** rejects equal performance on Iris (*p*=0.003), Breast Cancer (*p*=0.018), and Titanic (*p*=0.011)
- Bonferroni-corrected pairwise Wilcoxon tests: no individual pair reaches significance at α=0.05 after correction
- Effect sizes (Cohen's *d*): mostly small (< 0.5)

- [x] Statistical significance tested
- [x] All figure generation code released
- [x] Dependencies and versions stated
- [x] Runtime reported (< 5 min on standard laptop)

## Extending This Work

### Add a new dataset

```python
# data/loaders.py
def load_my_dataset():
    X, y = ...
    meta = dict(name="MyDataset", n_samples=..., n_features=..., n_classes=..., task="binary")
    return X, y, meta

ALL_DATASETS["my_dataset"] = load_my_dataset
```

### Add a new model

```python
# models/registry.py
def get_my_model():
    clf = MyClassifier(...)
    return _named(clf, "My Model")

ALL_MODELS["my_model"] = get_my_model
```

Then re-run:

```bash
python run_benchmark.py --models my_model --datasets iris wine
```
